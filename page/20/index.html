saltstack学习3-Grains和Pillar

2016-06-08
saltstack

三、 Grains和Pillar数据系统

(一) Grains

Grains 是静态信息，存储在minion端，在minion启动时收集 operating system, domain name, IP address, kernel, OS type, memory, and many other system properties等信息。如果 Grains 信息变化，需要重启 minion 收集，例如 IP 变化等。

作用：grains可用来在target中匹配minion或者state中使用。

可通过修改配置文件/etc/salt/minion添加自定义的grains，然后重启minion即可。
