__init__.py和__all__

2015-12-29
python
次阅读

文章目录

  • 1. 知识点
  • 2. 实验

1. 知识点

  1. from pkg import * 只能导入__init__.py 里 __all__ 指定的变量
  2. from pkg import * 不能导入单下划线开头的变量，除非单下划线开头的变量在 all 里
  3. 单下划线开头的变量可以指定全路径导入

[root@localhost Music]# cat pkg/__init__.py 
__all__ = ['kxn']
from pkg.pkg1.pkg1_a import kxn
_abc = 123

这个可以让用户直接从 pkg 导入其子包的变量，例如 from pkg import kxn

单下划线开始的变量，不能通过 from pkg import * 导入，但是可以指定全 路径导入，例如 from pkg.pkg1.pkg1_a import _bcd
