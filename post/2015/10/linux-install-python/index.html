安装Python 2.7.10和 3.5.0

2015-10-04
python
次阅读

同一个系统安装多个 python 版本，不影响系统自带版本。

1. 安装 Python 2.7.10

wget https://www.python.org/ftp/python/2.7.10/Python-2.7.10.tgz
tar xzvf Python-2.7.10.tgz
mkdir /usr/local/python27
cd Python-2.7.10
# 为了安装pyinstaller，建议如下configure
# ./configure --prefix=/usr/local/python27 --enable-shared
./configure --prefix=/usr/local/python27
make
make install
ln -s /usr/local/python27/bin/python /usr/bin/python27

2. 安装 Python 3.5.0

3.5.0 默认会安装 pip 工具，因为其它两个版本也安装了 pip ，所以需要添加软链接区分各个版本的 pip

wget https://www.python.org/ftp/python/3.5.0/Python-3.5.0.tgz
tar xzvf Python-3.5.0.tgz
mkdir /usr/local/python35
cd Python-3.5.0
# 为了安装pyinstaller，建议如下configure
# ./configure --prefix=/usr/local/python35 --enable-shared
./configure --prefix=/usr/local/python35
make
make install
ln -s /usr/local/python35/bin/python3.5 /usr/bin/python35
ln -s /usr/local/python35/bin/pip3.5 /usr/bin/pip35
# configure启用--enable-shared后执行
# cd /usr/local/python35/lib
# cp libpython3.5m.so.1.0 /usr/lib/
# cp libpython3.5m.so.1.0 /usr/lib64/
