理解了基础概念、变量使用等内容，接下来学习如何编写函数。

前面了解到当 Lisp 解释执行一个 List 时，首先看第一个 symbol 是否绑定了函数定义，或者说是否指向了一个函数定义。 如果一个 symbol 指向了一个函数定义，则称这个 symbol 为函数。

Function

用 Emacs Lisp 写的所有函数都是在其它函数的基础上编写的，除了一些 primitive function ，它们是用 C 语言编写的。 我们只需要了解有这么个概念即可，后面写 Emacs Lisp 函数也无需关心要调用的函数是不是用 C 语言写的。

defun

defen 是一个宏，是 define function 的缩写。

一个函数包含5个部分。

  1. 函数名

  2. 参数列表，没有参数的话使用空列表 ()

  3. 文档描述，可选部分

  4. interactive 部分，可选

  5. 函数体，描述函数具体执行的功能

上面提到的函数组成可以用如下模板表示。

(defun function-name (arguments…)
  "optional-documentation…"
  (interactive argument-passing-info)     ; optional
  body…)

如下是个完整的函数示例，第一部分是函数定义，第二部分是函数调用，分别在两部分的末尾执行 C-x C-e ，即可在屏幕下方看到函数执行结果。

  (defun multiply-by-seven (number)
