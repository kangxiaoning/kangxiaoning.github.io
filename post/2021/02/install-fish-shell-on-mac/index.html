记录在 Mac 上安装配置 fish 和 OMF 的过程。

安装Fish

安装 fish 并设置为默认shell。

1
2
3

brew install fish
echo "/usr/local/bin/fish" | sudo tee -a /etc/shells
chsh -s `which fish`

安装OMF

OMF是 Oh My Fish 的缩写，参考官方文档安装即可，我使用如下方法，因为使用 curl 时报错 curl (47) maximum (50) redirects followed ，为了快速使用 OMF 未花时间解决该问题。

1
2
3

git clone https://github.com/oh-my-fish/oh-my-fish
cd oh-my-fish
bin/install --offline

配置

环境变量

-U 可持久化配置，重启后也生效，具体可以通过 help set 查询。

1

set -U fish_user_paths $fish_user_paths /usr/local/opt/ruby/bin

技巧

  1. 补全

当你在 fish 里键入部分命令后， fish 会有一些提示给出来，补全技巧如下。

技巧描述
目录补全Tab
命令补全->
部分补全Alt + ->

插件

记录一些有助于提升效率的插件。

fzf

从 zsh + Oh My Zsh 切换过来，经常使用的 Ctrl + r 搜索历史命令在 fish 默认不可用，可以通过 fzf 插件来支持。

1
2

brew install fzf
omf install fzf

autojump

install

1
2

brew install autojump
omf install autojump

注意在安装结束按提示配置 fish ，如果没有提示，可重新执行 brew reinstall autojump ，如下所示。
