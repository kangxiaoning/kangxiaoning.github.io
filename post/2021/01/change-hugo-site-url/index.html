修改博客URL格式

2021-01-19

times read

之前博客的URL里日期和文件名是连在一起的，写了个脚本，将年、月分割出来作为目录，这样生成的博客URL年、月是独立的path，脚本如下。

#!/bin/bash

cd ~/workspace/blog/content/post
for f in $(ls *.md)
do
    year=$(echo $f | cut -d - -f 1)
    month=$(echo $f | cut -d - -f 2)
    filename=${f:11}
    mv $f $filename
    directory=$year/$month/
    if [ ! -d $directory ]; then
        mkdir -p $directory
    fi
    mv -v $filename $directory
done
