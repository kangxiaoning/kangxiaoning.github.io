1. 介绍

string模块包含一些有用的常量和类。

常量主要是一些数字，大小写字母，标点符号和可打印字符。

类主要有Formatter和Template，Formatter和str.format差不多，有一些自定义的format行为， Template可以在字符串嵌入变更，然后使用Template类的substitute方面地替换变量。

规则如下

  1. $$ is an escape; it is replaced with a single $.
  2. $identifier names a substitution placeholder matching a mapping key of “identifier”. By default, “identifier” must spell a Python identifier. The first non-identifier character after the $ character terminates this placeholder specification.
  3. ${identifier} is equivalent to $identifier. It is required when valid identifier characters follow the placeholder but are not part of the placeholder, such as “${noun}ification”.

3. string.Template

可以方便的生成一些linux管道命令，如下awk命令中对$作了转义。

1
2
3
4

>>> cmd = string.Template("ifconfig $interface | grep HWaddr | awk '{print $$NF}'")
>>> cmd.substitute(interface='eth0')
"ifconfig eth0 | grep HWaddr | awk '{print $NF}'"
