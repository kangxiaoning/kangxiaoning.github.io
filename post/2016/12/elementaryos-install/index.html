突然想体验下号称最漂亮的Linux - elementaryOS，安装过程很简单，主要记录下一些常用软件的安装过程。

总结

总的来说，界面还是挺漂亮的，使用上也还OK。

  1. 最小化 - 在 Dock 上点击图标
  2. 安装软件 - 启动后会自动记录到“应用程序”里，可以创建快捷方式到 Dock
  3. 自带的浏览器感觉不好用
  4. 在VMware Workstation上安装，需要手动挂载cdrom安装vmtools

1. 安装vmtools

物理机操作

VMware Workstation –> 虚拟机 –> 安装 VMware Tools

虚拟机操作

sudo mkdir /mnt/cdrom
sudo mount -t auto /dev/cdrom /mnt/cdrom
cd /mnt/cdrom/
cp VMwareTools-10.0.10-4301679.tar.gz /tmp/
cd /tmp/
sudo tar xzvf VMwareTools-10.0.10-4301679.tar.gz
cd vmware-tools-distrib/
sudo ./vmware-install.pl -d

4. 安装五笔拼音输入法

执行如下命令安装，完成后重启会在任务栏出现输入法图标，然后设置下，删除不需要的输入法即可。

1

sudo apt-get install fcitx fcitx-table-wbpy fcitx-config-gtk
