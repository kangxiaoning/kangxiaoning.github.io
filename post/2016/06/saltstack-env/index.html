一、 实验环境

(一) 资源规划

角色IP
Master10.20.19.100
minion10.20.19.101
minion10.20.19.102
minion10.20.19.103
minion10.20.19.105

(二) Linux平台安装

1. Master安装

1
2

yum -y install salt-master
chkconfig salt-master on

2. Minion安装

1
2

yum -y install salt-minion
chkconfig salt-minion on

(三) Windows平台安装

1. Windows 2008

需要先安装Microso Visual C++ 2008 x64 SP1再安装salt-minion，否则启动过程会出现如下提示。

2. Windows 2012

直接安装salt-minion即可。

(四) 配置

最小化配置，只需要在minion配置文件指定master IP/域名即可。

1
2
3

[root@ABC-L0008181 ~]# more /etc/salt/minion|grep -v ^#|grep -v ^$
master: 10.20.19.100
[root@ABC-L0008181 ~]#

(五) 启动

1
2
3

service salt-master start
service salt-minion start
