1. ChainMap

  • ChainMap 是 3.3 版本新增
  • 支持所有 dict 方法

dict-like class for creating a single view of multiple mappings.

是 dict-like 容器，对 dict 进行了封装，可以连接多个mapping提供一个单独的view，使之 使用起来就是一个dict一样。将多个mapping连接后存储在一个list中，因此对新容器访问 是有序的,所有的增删改查操作都仅操作ChainMap容器中的第一个mapping。

使用场景

如下假设 default_env 存放默认配置，custom_env 存放用户自定义的配置，我们使用 ChainMap 将两个 dict 连接起来，custom_env 放在前面，这样如果用户提供了配置， 我们就使用用户的配置，如果用户没提供配置就使用默认配置。

2. Counter

  • 统计hashable objects中元素的个数
  • 底层实现是 dict
  • 提供dict的接口
  • 支持 + - & | 运算
1

>>> c = Counter()                           # a new, empty counter
>>> c = Counter('gallahad')                 # a new counter from an iterable
>>> c = Counter({'red': 4, 'blue': 2})      # a new counter from a mapping
>>> c = Counter(cats=4, dogs=8)             # a new counter from keyword args

使用场景

统计一句话中出现最多的前N个单词。

Counter可以很简单解决这样的问题，如下我们将句子split为list，然后送给Counter 计算，直接用most_common方法返回前N个最多的单词。

3. defaultdict

通常一个dict就是一个key对应一个value的映射，如果想一个key对应多个value，那 么需要将这多个value放到另外的容器中比如list、dict、set、tuple等，例如如果 有序就是list/tuple，如果消除重启就是set/dict等。

The defaultdict is a subclass of Python’s dict that accepts a default_factory as its primary argument. The default_factory is usually a Python type, such as int or list, but you can also use a function or a lambda too.

  • subclass of Python’s dict
  • 自动初始化key刚开始的值
  • 指定key对应的多个value要使用的容器类型
  • 支持自定义默认值

使用场景

常用的场景如下，第一种是自己实现，第二种是使用defaultdict，可以看到简单了许多。

d = {}
for key, value in pairs:
    if key not in d:
        d[key] = []
    d[key].append(value)

d = defaultdict(list)
for key, value in pairs:
    d[key].append(value)

4. deque

  • double-ended queue
  • a replacement container for the Python list

使用deque(maxlen=N) 构造函数会新建一个固定大小的队列。当新的元素加入并 且这个队列已满的时候，最老的元素会自动被移除掉。

使用场景

保留最后N 个元素，比如可以基于 deque 实现简单的tail命令。

#!/usr/bin/python

import sys
import argparse
from collections import deque

def positive_int(value):
    ivalue = int(value)
    if ivalue <= 0:
         raise argparse.ArgumentTypeError("%s is an invalid positive int value" % value)
    return ivalue

def get_last():
    """Returns the last n lines from the file."""
    parser = argparse.ArgumentParser(
        description="A command like tail on Linux.",
        epilog="""Example:
                    ./tail.py FILENAME -n N
               """
    )

    parser.add_argument('filename', help="Filename", action="store")
    parser.add_argument('-n', help="Number (default is 5)", type=positive_int, default=5)
    args = parser.parse_args()
    filename = args.filename
    n = args.n
    try:
        with open(filename) as f:
            lines = deque(f, n)
        for line in lines:
            print(line),
    except OSError:
        print("Error opening file: {}".format(filename))
        raise


if __name__ == '__main__':
    get_last()
