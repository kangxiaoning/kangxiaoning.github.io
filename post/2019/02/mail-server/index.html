术语

SMTP (Simple Mail Transfer Protocol)

发送电子邮件，需要遵循一定的通讯规则，SMTP 协议就是用来定义这种通讯规则的。

POP3（Post Office Protocol 3)

主要用于支持使用客户端远程管理在服务器上的电子邮件。该协议是在 RFC-1939 中定义的，是 Internet 上的大多数人用来接收邮件的机制。POP3 采用 Client/Server 工作模式，默认使用 TCP 110 端口。

IMAP (Internet Message Access Protocol)

相对于 pop3 协议所有邮件的管理都需要下载下来进而管理，IMAP 提供了用户远程访问邮件服务器的途径，因而通过 IMAP 用户可以直接管理邮件服务器上的邮件，比如创建目录，分类管理等。

MUA (Mail User Agent)

邮件客户端，即写邮件和收邮件的客户端，比如 foxmail、outlook、网页邮箱等。

MTA (Mail Transfer Agent)

负责收邮件和转发邮件的服务，是邮件服务的核心，使用 SMTP 协议，一般提到 mail server 也是指 MTA，比如 exchange、sendmail、postfix、qmail 等。

MDA (Mail Delivery Agent)

对 MTA 收到的邮件进行分析，决定是保存下来还是继续转发，如果 MTA 收到的邮件目标是自己则由 MDA 保存到对应的存储空间，如果是其它 MTA 则要转发。 有些 MTA 集成 MDA 功能，比如 exchange。

relay

使用不同的邮件服务会涉及到 MTA 转发，由一台 MTA 转往另一台 MTA 就称为 relay。
