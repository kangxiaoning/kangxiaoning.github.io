解决方法

  1. 在生产确认下该文件不存在
  2. 因为不存在，所以在控制文件将该文件记录删除即可，执行如下命令
  3. alter database datafile 13 offline drop;
  4. alter database recover managed standby database disconnect parallel 4;
  5. 观察同步已经正常

分析总结

由于该文件在生产已经不存在，但是DG端的控制文件还有记录，因此导致无法同步，处理方法是在DG端的控制文件删除该文件记录即可。
