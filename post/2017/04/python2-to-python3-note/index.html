1. String

python2.6

  1. str默认是ASCII，不可变对象
  2. ASCII用str表示，bytes是str的同义词，不可变对象
  3. Unicode必须用u前缀（u’中'）表示，type是unicode，不可变对象
  4. bytearray元素赋值可用str/integer，bytearray初始化用str

python3.5

  1. str默认是Unicode，不可变对象
  2. ASCII用bytes表示，不可变对象
  3. Unicode不需要u前缀，type是str，不可变对象
  4. bytearray元素赋值必须用integer，bytearray初始化用bytes

2. json

python2.6

  1. json.dumps支持指定编码，如json.dumps(encoding=‘utf-8’)
  2. json.dumps支持输入Unicode(unicode类型)和bytes(str类型)
  3. json.dumps返回str/unicode类型

python3.5

  1. json.dumps不支持指定编码
  2. json.dumps支持输入Unicode(str类型)，不支持bytes(bytes类型)
  3. json.dumps返回bytes类型

3. base64

python2.6

  1. 支持base64.encodestring/decodestring

python3.5

  1. 废弃base64.encodestring/decodestring，base64.encodebytes/decodebytes替代

4. pickle

变化：在python3.5，open文件时需要指定’rb’或者’wb'。

python2.6

  1. 如下代码使用正常
1
2

with open('init.txt') as f:
    init_data = pickle.load(f)

python3.5

  1. 代码修改成如下
1
2

with open('init.txt', 'rb') as f:
    init_data = pickle.load(f)
