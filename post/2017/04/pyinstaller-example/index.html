1.1 import 处理

如果代码里使用了动态import，pyinstaller无法找到对应的依赖，需要明确指出，下面示例通过hiddenimports来指出需要显示引入的依赖。

如下打包将python代码打包成单个文件，会在当前目录生成dist/build目录，可执行文件在dist中，该文件可用来发布给其它人，无需安装python，但是每个平台需要打包一个文件。
