ubuntu 16.04安装有道词典

2017-05-04
Linux
次阅读

文章目录

有道官方Ubuntu版本的的deb包由于gstreamer0.10-plugins-ugly依赖，无法成功安装，实际在 16.04版本并不需要该依赖，所以只需要把该依赖检查删除再安装就可以了。

  1. 下载

youdao-dict_1.1.0-0-ubuntu_amd64.deb

  1. 处理依赖并安装

# 解压.deb到youdao-dict目录
sudo dpkg -X ./youdao-dict_1.1.0-0-ubuntu_amd64.deb youdao-dict

# 将control文件解压到youdao-dict/DEBIAN目录下
sudo dpkg -e ./youdao-dict_1.1.0-0-ubuntu_amd64.deb youdao-dict/DEBIAN

# 编辑control文件，在Depends里删除gstreamer0.10-plugins-ugly
sudo vim youdao-dict/DEBIAN/control

# build新的软件包，名字为youdaobuild.deb
sudo dpkg-deb -b youdao-dict youdaobuild.deb

# 安装新build的包
sudo dpkg -i youdaobuild.deb

# 安装依赖
sudo apt install -f

# 重新安装youdao成功
sudo dpkg -i youdaobuild.deb
