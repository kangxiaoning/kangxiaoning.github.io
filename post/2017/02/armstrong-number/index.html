1. 水仙花数

水仙花数（Narcissistic number）也被称为超完全数字不变数（pluperfect digital invariant, PPDI）、自恋数、自幂数、阿姆斯壮数或阿姆斯特朗数 （Armstrong number），水仙花数是指一个 n 位数（n≥3 ），它的每个位上的数字 的 n 次幂之和等于它本身（例如：1^3 + 5^3+ 3^3 = 153）。

2. 问题

求 100~999之间所有的水仙花数。

3. 解题思路

  1. 编写函数是否为水仙花数
  2. 在某一范围内的数检测是否为水仙花数

判断是否为水仙花数，需要得到给定数字的每一位，然后求N次方即可。

通过如下规律可知，给任意数字模10总能得到最后一位，除以10取整总能得到将数缩短一位。

因此可以对一个数字不断取模并缩短一位，最终可以得到第一位数字。

# -*- coding:utf-8 -*-

def is_armstrong_number(number):
    digits = []
    tmp_number = number
    while tmp_number > 0:
        digits.append(tmp_number % 10)
        tmp_number /= 10
    exponential = len(digits) # 指数为数字的位数
    # 列表表达式生成每个位上的数字的 n 次幂，结果为list
    # sum对所有数字位N次幂求各
    return sum([d ** exponential for d in digits]) == number

for n in xrange(100, 1000):
    if is_armstrong_number(n):
        print(n)
