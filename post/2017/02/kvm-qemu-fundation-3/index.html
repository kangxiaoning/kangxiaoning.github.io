1. 实验环境

通过在vmware workstation上启动centos 7的虚拟机，以该虚拟机作为HOST进行KVM 实验，以一个精简的linux作为guest的OS镜像启动KVM虚拟机。

经过实验发现，在vmware workstation上如果不打开“虚拟化CPU性能计数器”，在启 动kvm的vm时会导致host（vmw虚拟的）死机，卡在CPU计数器那里。

#!/bin/bash
#
bridge=br0

if [ -n "$1" ]; then
    ip link set $1 up
    brctl addif $bridge $1
    [ $? -eq 0 ] && exit 0 || exit 1
esle
    echo "Error: no interface specified."
